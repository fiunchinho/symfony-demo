# Symfony Demo Application

## Installation

### Locally
Clone this repository and run the following commands

```
$ sudo apt-get install -y sqlite php5-sqlite
$ git clone https://github.com/fiunchinho/symfony-demo
$ cd symfony-demo/
$ composer install
```
